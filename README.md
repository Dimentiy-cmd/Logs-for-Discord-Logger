# Logs for Discord Logger 1.2

## Русский

### Описание

*Logs for Discord Logger* — это сайт бота для Discord, обсепечивающий вывод логов от бота Discord Logger. Ссылка на гит хаб бота - https://github.com/Dimentiy-cmd/DiscordLoggerBot

### Настройка базы данных

Чтобы подключить наш сайт к базе данных, в файле config.php в папке include изменяем данные на те, что были в боте и сохраняем.

### Установка

В своем гайде я покажу как поставить сайт на хостинг [MAZE-HOST](https://maze-host.ru/).

Для начала перейдите в панель управления сайтом, а затем перейдите в ISP Manager:

![image](https://cdn.imgchest.com/files/4nec8ej9mo4.PNG)

По умолчанию у вас будет бесплатный домен, я покажу деплой на его примере. В менеджере файлов в каталоге www переходим в папку с вашим доменом. 

![image](https://cdn.imgchest.com/files/7ogcblnlp9y.png)

В этой папке удаляем файл index.html и загружаем файлы нашего сайта. После того как закинули файлы можем перейти на наш сайт. Чтобы получить доступ к сайту входим по следующим данным: имя пользователя - root, пароль - root. Эти данные можно изменить в базе данных.
